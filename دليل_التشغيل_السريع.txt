┌────────────────────────────────────────────────────────────────────┐
│                          📋 المتطلبات                              │
└────────────────────────────────────────────────────────────────────┘

    1. Docker Desktop (يجب أن يكون مُفعّل ويعمل)
    2. متصفح (Chrome, Edge, Firefox)
    3. 4GB RAM على الأقل

┌────────────────────────────────────────────────────────────────────┐
│                    🚀 طريقة التشغيل (3 خطوات)                      │
└────────────────────────────────────────────────────────────────────┘

    الخطوة 1: افتح Docker Desktop
    ─────────────────────────────────
    • تأكد أن Docker Desktop يعمل (أيقونة في شريط المهام)

    الخطوة 2: شغّل النظام
    ─────────────────────────
    • انقر نقراً مزدوجاً على: start_system.bat
    • أو استخدم الاختصار: "نظام السند - تشغيل" من سطح المكتب

    الخطوة 3: افتح المتصفح
    ──────────────────────────
    • العنوان: http://localhost
    • أو: http://localhost:3000

    بيانات الدخول:
    ──────────────────────────
    • اسم المستخدم: admin
    • كلمة المرور: admin123

┌────────────────────────────────────────────────────────────────────┐
│                    🎮 أوامر التحكم السريع                          │
└────────────────────────────────────────────────────────────────────┘

    ┌──────────────────────────────────────────────────────────┐
    │  الملف                  │  الوظيفة                     │
    ├──────────────────────────────────────────────────────────┤
    │  start_system.bat       │  تشغيل النظام كامل           │
    │  stop_system.bat        │  إيقاف النظام كامل           │
    │  restart_system.bat     │  إعادة تشغيل النظام          │
    │  status_system.bat      │  فحص حالة النظام             │
    │  logs_system.bat        │  عرض سجلات النظام            │
    └──────────────────────────────────────────────────────────┘

    💡 نصيحة: استخدم start_system.bat و stop_system.bat فقط
       فهما يكفيان لمعظم الاستخدامات

┌────────────────────────────────────────────────────────────────────┐
│                    🔗 عناوين الوصول للنظام                        │
└────────────────────────────────────────────────────────────────────┘

    الواجهة الرئيسية (التطبيق):
    • http://localhost
    • http://localhost:3000

    الواجهة الخلفية (API):
    • http://localhost:8000
    • التوثيق: http://localhost:8000/docs

    قاعدة البيانات (PostgreSQL):
    • المنفذ: 5433
    • يمكن الوصول عبر pgAdmin أو DBeaver

┌────────────────────────────────────────────────────────────────────┐
│                    ⚠️  استكشاف الأخطاء                             │
└────────────────────────────────────────────────────────────────────┘

    المشكلة: النظام لا يعمل بعد التشغيل
    ────────────────────────────────────────
    الحل:
    1. تأكد أن Docker Desktop يعمل
    2. شغّل status_system.bat لرؤية الحالة
    3. شغّل restart_system.bat لإعادة التشغيل

    المشكلة: لا أستطيع الدخول
    ───────────────────────────
    الحل:
    • اسم المستخدم: admin
    • كلمة المرور: admin123

    المشكلة: خطأ "port is already allocated"
    ─────────────────────────────────────────
    الحل:
    • تأكد أن النظام لا يعمل مرتين
    • شغّل stop_system.bat أولاً

┌────────────────────────────────────────────────────────────────────┐
│                    📞 الدعم والمساعدة                             │
└────────────────────────────────────────────────────────────────────┘

    إذا واجهت مشكلة:
    1. شغّل status_system.bat - يعرض حالة كل خدمة
    2. شغّل logs_system.bat - يعرض السجلات والأخطاء
    3. أعد تشغيل النظام: stop_system.bat ثم start_system.bat

┌────────────────────────────────────────────────────────────────────┐
│                    💡 نصائح مهمة                                  │
└────────────────────────────────────────────────────────────────────┘

    ✓ دائماً أوقف النظام (stop_system.bat) قبل إيقاف الكمبيوتر
    ✓ استخدم status_system.bat للتأكد من حالة النظام
    ✓ لا تغلق نافذة start_system.bat أثناء التشغيل
    ✓ انتظر حتى تظهر رسالة "تم التشغيل بنجاح" قبل فتح المتصفح
